Componente Animado - Figura Bailarina

Este componente es un pequeño personaje animado en SVG que puede utilizarse como parte visual de un footer. Utiliza animaciones CSS para crear un efecto de baile.

                <line x1="12" y1="7.5" x2="12" y2="14" stroke="#aaaaaa" stroke-width="1.2"/>
